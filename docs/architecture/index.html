Strukturübersicht

Die Architekturspezifikation definiert den gemeinsamen Rahmen, in dem alle Building Blocks operieren. Sie ist normativ für alle Komponenten des Stacks und beschreibt Mindestanforderungen für Stack-Konformität.

Normativität
Alle Anforderungen folgen RFC 2119. MUSS ist verpflichtend für Stack-Konformität. SOLL ist empfohlen, Abweichungen erfordern Begründung. KANN ist optional.
